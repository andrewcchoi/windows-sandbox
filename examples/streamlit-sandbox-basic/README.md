# Streamlit Sandbox (Basic Mode)

**Self-contained** Streamlit app demonstrating what the sandbox-maxxing plugin generates in **Basic mode**.

## What is Basic Mode?

Basic mode is the quickest way to get started:
- **Minimal questions** (1-2): Auto-detects your project type
- **Sensible defaults**: Strict firewall, minimal VS Code extensions
- **Fast setup**: Generates all configs in one shot

## Features

### Application Features
- **Streamlit App**: Interactive connection testing interface
- **Database Integration**: PostgreSQL connection and query execution
- **Caching Layer**: Redis connection and key-value operations
- **Visual Feedback**: Real-time status indicators for each service

### DevContainer Features (Basic Mode)
- **Auto-detected stack**: Python 3.12 with Streamlit
- **Database services**: PostgreSQL + Redis
- **Network security**: Strict firewall with domain allowlist
- **VS Code extensions**: Minimal (Python extension only)
- **Port forwarding**: Streamlit (8501), PostgreSQL (5432), Redis (6379)
- **Quick setup**: 1-2 minutes from start to running container

## What's Included

Generated by `/sandbox:basic`:

- **`.devcontainer/devcontainer.json`** - Minimal DevContainer config with Python extension
- **`.devcontainer/Dockerfile`** - Flexible Python 3.12 image
- **`.devcontainer/init-firewall.sh`** - Strict firewall (whitelist mode)
- **`docker-compose.yml`** - PostgreSQL + Redis services
- **`app.py`** - Streamlit connection test app
- **`requirements.txt`** - Python dependencies

## Usage

### Open in VS Code DevContainer

```bash
# Clone or copy this directory
cd streamlit-sandbox-basic

# Open in VS Code
code .

# Reopen in Container
# Press F1 â†’ "Dev Containers: Reopen in Container"

# Install Claude Code (required after each container rebuild)
curl -fsSL https://claude.ai/install.sh | sh
```

> **Note:** Claude Code must be reinstalled each time the devcontainer is rebuilt. If you're in an offline or air-gapped environment where the installation script cannot be reached, see [TROUBLESHOOTING.md](../../docs/TROUBLESHOOTING.md#claude-code-installation) for alternative installation methods.

### Run the App

```bash
# Inside the container
streamlit run app.py
```

Open http://localhost:8501 and click the test buttons to verify connectivity.

## Troubleshooting

**"Connection failed" errors:**
1. Ensure Docker services are running: `docker compose ps`
2. Check services are healthy: `docker compose ps` (look for "healthy" status)
3. Restart services: `docker compose restart postgres redis`
4. Check logs: `docker compose logs postgres redis`

**Port already in use:**
- Streamlit default port is 8501
- Change with: `streamlit run app.py --server.port 8502`
- Or stop conflicting service: `lsof -ti:8501 | xargs kill -9`

**Claude Code installation issues:**
- See [TROUBLESHOOTING.md](../../docs/TROUBLESHOOTING.md#claude-code-installation) for offline installation methods
- Verify installation: `claude --version`

**Services not starting:**
- Check Docker Desktop is running
- Verify disk space: `docker system df`
- Rebuild container: Press F1 â†’ "Dev Containers: Rebuild Container"

For more help, see [../../docs/TROUBLESHOOTING.md](../../docs/TROUBLESHOOTING.md)
