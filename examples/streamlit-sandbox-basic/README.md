# Streamlit Sandbox (Basic Mode)

**Self-contained** Streamlit app demonstrating what the windows-sandbox plugin generates in **Basic mode**.

## What is Basic Mode?

Basic mode is the quickest way to get started:
- **Minimal questions** (1-2): Auto-detects your project type
- **Sensible defaults**: Strict firewall, minimal VS Code extensions
- **Fast setup**: Generates all configs in one shot

## What's Included

Generated by `/sandbox:basic`:

- **`.devcontainer/devcontainer.json`** - Minimal DevContainer config with Python extension
- **`.devcontainer/Dockerfile`** - Flexible Python 3.12 image
- **`.devcontainer/init-firewall.sh`** - Strict firewall (whitelist mode)
- **`docker-compose.yml`** - PostgreSQL + Redis services
- **`app.py`** - Streamlit connection test app
- **`requirements.txt`** - Python dependencies

## Basic Mode Characteristics

| Feature            | Basic Mode                                        |
| ------------------ | ------------------------------------------------- |
| Questions          | 1-2 (auto-detected)                               |
| Dockerfile         | Flexible template (works with any Python project) |
| VS Code Extensions | 1 (ms-python.python)                              |
| Firewall           | Strict (default)                                  |
| Services           | Auto-detected (PostgreSQL + Redis)                |
| Explanations       | None (just works)                                 |

## Compare with Other Modes

- **Advanced mode**: 5-7 questions, configurable Dockerfile with build args, 5 VS Code extensions
- **YOLO mode**: 10-15+ questions, technology-optimized Dockerfile, 11 extensions, production-ready configs

See `examples/demo-app-sandbox-advanced/` and `examples/demo-app-sandbox-yolo/` for comparisons.
